# 🌱 Bloom With You

A modern, responsive web application celebrating personal growth, community, and meaningful connections.

## What is Bloom With You?

Bloom With You is a platform dedicated to helping people grow together. Whether you're pursuing career advancement, personal development, or building real connections with others on the same journey—we're here to support your growth.

## Features

✨ **Modern Design** - Clean, professional interface with a green accent color scheme
🚀 **Responsive** - Fully responsive design that works perfectly on all devices
🌱 **Community-Focused** - Built around the idea of growing together
💪 **Engagement** - Multi-section layout with clear calls-to-action
🎨 **Polished** - Smooth animations and professional styling

## Sections

- **Hero** - Powerful opening statement with clear CTAs
- **Stats** - Key metrics showing impact and growth
- **About** - Mission statement and core values
- **How It Works** - Four-pillar approach (Learn, Connect, Track, Achieve)
- **Why Choose Us** - Differentiators and value propositions
- **Stories** - Success stories from the community
- **Contact** - Multiple contact options and enrollment form

## Tech Stack

- **React 19** - Modern UI framework
- **CSS3** - Custom styling with animations
- **Responsive Design** - Mobile-first approach
- **Modern Color Scheme** - Green accent (#22c55e) on dark navy background

## Installation

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

The optimized production build will be in the `build/` directory.

## Deployment

This site is ready to deploy to Vercel, Netlify, or any static hosting service.

### Vercel (Recommended)

```bash
npm install -g vercel
vercel
```

### Other Platforms

The build folder can be deployed directly to any static hosting service. Just ensure the homepage is set correctly in `package.json`.
